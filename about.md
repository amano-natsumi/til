## 変数名

・方法、計算 ではなく もの、問題を表す単語にする
・装飾子は後ろにつける
numは使用しない

```
count index
合計: total
begin/end
first/last
locked/unlocked
min/max
next/previous
old/new
opened/closed
visible/invisible
source/target
source/destination
up/down
```

・bool変数

```
done
error
found
success/ok
```
